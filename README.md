# C_Compiler

Um compilador simples para a linguagem C.

## Pré-requisitos

Para compilar e executar este projeto, você precisará instalar as seguintes dependências:


```bash
sudo apt install flex bison
```

## Compilação

```bash
make
```

## Teste

```bash
make test-lex
```

```bash
make test-bison
```

## Limpeza

```bash
make clean
```
